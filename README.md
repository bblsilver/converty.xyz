Converty.xyz is a completely client-sided image converter, allowing users to quickly convert images into various popular image formats instantly for download. 

## Features
* **100% Secure & Private:** All conversions happen directly in your browser using the HTML5 Canvas API. Your images never touch a server.
* **Modern Interface:** A clean, minimal design featuring a dynamic pastel-red user interface.
* **Sleek Interactions:** Clean dropdown menu transitions that smoothly reveal formatting options on hover.
* **Format Support:** Convert your files instantly to **JPG, PNG, WebP, GIF, or BMP**.
* **Smart Processing:** Automatically adds a clean white background background when converting transparent images to JPG to prevent rendering artifacts.
